# Практична робота "Створення і використання статичних методів"

## Завдання №17: Шифрування шифром XOR

### Опис завдання
Метою роботи є реалізація методу `encrypt` у класі `Exercise` для шифрування заданої фрази за допомогою алгоритму **XOR (виключне АБО)** з використанням слова-ключа.

> **Шифр XOR** — це алгоритм адитивного шифрування, де кожен символ відкритого тексту поєднується із символом ключа за допомогою побітової операції XOR. Особливістю алгоритму є те, що застосування тієї ж операції до зашифрованого тексту повертає початковий текст.

### Деталі реалізації
У класі `Exercise` створено статичний метод, який виконує наступні дії:
1.  Приймає вхідний рядок (`text`) та ключ шифрування (`key`).
2.  Проходить циклом по кожному символу тексту.
3.  Виконує операцію `^` (XOR) між символом тексту та відповідним символом ключа.
4.  Для зациклення ключа (якщо він коротший за текст) використовується оператор остачі від ділення: `key.charAt(i % key.length())`.
